# Vivek Kushwaha Portfolio

A production-ready, modern personal portfolio website built with Next.js 15, React, Tailwind CSS, and Framer Motion.

## Features

- **Dark Theme:** Sleek and professional appearance by default.
- **Modern UI:** Glassmorphism effects, clean typography, and subtle gradients.
- **Responsive:** Mobile-first design that works perfectly on all devices.
- **Animated:** Smooth scroll-reveal and hover animations using Framer Motion.
- **Performance:** Optimized for speed using Next.js Server Components.
- **SEO Ready:** Proper metadata and semantic HTML.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS (v4)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Language:** JavaScript (React.js)

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open in browser:**
    Navigate to [http://localhost:3000](http://localhost:3000)

## Folder Structure

- `app/`: Contains the App Router pages and layouts.
- `app/components/`: Reusable UI components and page sections.
- `app/data/`: Portfolio content (edit `portfolio.js` to change info).
- `app/lib/`: Utility functions.
- `public/`: Static assets (images, icons, etc.).

## Customization

To update the content, simply modify the `app/data/portfolio.js` file. It contains all the text, links, and skill lists shown on the website.
